Project Goal

Our project focuses on one main question:

Which customer and campaign factors are most useful for predicting whether a customer converts?

We use the dataset to study customer demographics, marketing channels, engagement behavior, and campaign performance.

Dataset Overview(Yushan Huang)

  • Rows: 8,000 customer records
  • Columns: 20 variables
  • Target variable: Conversion
  • Data type: Synthetic digital marketing campaign data
  • File location in repo: data/digital_marketing_campaign_dataset.csv

Prediction Plan(Sora Hosoda)

  1. Clean and inspect the dataset
  2. Explore conversion patterns by customer and campaign variables
  3. Split data into training and testing sets
  4. Build a classification model to predict Conversion
  5. Evaluate performance using accuracy, precision, recall, and confusion matrix

Expected Insights(Sora Hosoda)

By analyzing this dataset, we expect to identify:

  • Which customer groups are more likely to convert
  • Which campaign channels perform better
  • How engagement metrics relate to conversion
  • Which variables are useful for prediction
